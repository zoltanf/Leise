A careful performance test should be simple to repeat and easy to inspect. The speaker reads each sentence at a steady pace, with ordinary punctuation and familiar vocabulary. We measure model preparation separately from transcription, then compare the first run with later runs after the model is warm. Stable fixtures help reveal real changes in latency without confusing them with different words, microphones, or background noise. Every result includes the source hash, audio duration, selected model, and exact software environment.
